# Soul Map Navigator (Елизабета)

Vite + React + Tailwind + shadcn/ui website.

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy to Vercel (new project)

- Push this folder to a new GitHub repo (or GitLab/Bitbucket).
- In Vercel click **Add New → Project**, import the repo.
- **Framework preset**: `Vite`
- **Build command**: `npm run build`
- **Output directory**: `dist`
- Deploy.

Notes:
- `vercel.json` is included so React Router routes work on refresh.
